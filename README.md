## Portfolio highlights

- **Project A - Simplified Java-like language compiler**  
  **Purpose:** Compile a simplified Java-like language into bytecode to demonstrate lexing, parsing, semantic analysis, and code generation.  
  **Tech:** Java, Lex, Cup.  
  **Key result:** End-to-end compiler that successfully compiles and runs sample programs, validating the full compilation pipeline.  
  **Repo:** https://github.com/Zerato7/MJCompiler

- **Project B - Web app for mountain cottage booking**  
  **Purpose:** Provide a web-based system to list cottages, check availability, and manage reservations.  
  **Tech:** Spring Boot, Angular.  
  **Key result:** End-to-end booking workflow with a responsive frontend and Java backend.  
  **Repo:** https://github.com/Zerato7/mountain-cottages-reservations

- **Project C - Operating system kernel for RISC‑V**  
  **Purpose:** Implement a minimal operating system kernel targeting the RISC‑V architecture to explore low-level OS concepts.  
  **Tech:** C/C++, Assembly, Makefile.  
  **Key result:** Minimal RISC‑V kernel that demonstrates core kernel functionality.  
  **Repo:** https://github.com/Zerato7/riscV-kernel
